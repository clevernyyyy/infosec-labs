Welcome to Xtreme Sender Service!

Objective:

The goal of this lab is to use cross-site-scripting (xss) to steal credits from the megarich user by sending a message from the pooruser.

There are two users on this box, both vulnerable to xss attacks. You'll need to open two browsers (or a private session), to log into both users. The users are listed below:

User Password
pooruser pass
megarich abc123

Stretch Goals:

While the main goal is to steal credits, there are several other fun things that you can do, including:

  • Can you login as the megarich user? How would you go about that? Do you steal a session? Social Engineer the password?
  • How would you go about stealing all of the megarich user's messages? All of his messages are just displayed on the client side html, how can you grab it?
  • Clearly the data is being saved somewhere, is it possible to play with the data?
